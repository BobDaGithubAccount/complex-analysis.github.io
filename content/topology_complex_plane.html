Topology of the complex plane

An $\epsilon$ \textbf{neighbourhood},\index{neighbourhood} also called \textbf{open ball}, of a complex number $z_0$ consists of all points $z$ lying inside but not on a circle centred at $z_0$ and with radius $\epsilon>0$ and is expressed by \begin{eqnarray} B_{\epsilon}(z_0)=\{z:|z-z_0|<\epsilon\} \end{eqnarray} The \textbf{closed} $\epsilon$ \textbf{neighbourhood}\index{neighbourhood!closed} of $z_0$ is expressed by \begin{eqnarray} \overline{B}_{\epsilon}(z_0)=\{z:|z-z_0|\leq \epsilon\} \end{eqnarray} And finally, a \textbf{deleted} $\epsilon$ \textbf{neighbourhood}\index{neighbourhood!deleted} of $z_0$ is expressed by \begin{eqnarray} B_{\epsilon}(z_0)\setminus \{z_0\}=\{z:0<|z-z_0|< \epsilon\} \end{eqnarray} Figure \ref{balls} shows the geometrical representation of the following examples:\vspace{-0.3cm} \begin{enumerate} \item $B_{1}(0)=\{z:|z|<1\}$ \item $\overline{B}_{\frac{7}{8}}\left(-1-\sqrt{2}i\right)=\left\{z:|z-(-1-\sqrt{2}i)|\leq \frac{7}{8}\right\}$ \item $\overline{B}_{\frac{1}{2}}\left(2+\sqrt{3}i\right)\setminus \left\{2+\sqrt{3}i\right\}=\left\{z:0<|z-(2+\sqrt{3}i)|\leq\frac{1}{2}\right\}$ \end{enumerate} \vspace{-0.3cm} \begin{figure}[h] \centering \includegraphics[width=0.5\textwidth]{balls} \caption{Geometrical representation of neighbourhoods.}\label{balls} \end{figure} \newpage \textbf{Classification of points} A point $z_0$ is said to be an \textbf{interior point}\index{point!interior} of a set $S\subset \mathbb C$ whenever there is some neighbourhood of $z_0$ that contains only points of $S$; it is called an \textbf{exterior point}\index{point!exterior} of $S$ when there exists a neighbourhood of it containing no points of $S$. If $z_0$ is neither of these, it is a \textbf{boundary point}\index{point!boundary} of $S$. A boundary point is, therefore, a point all of whose neighbourhoods contain at least one point in $S$ and at least one point not in $S$. The totality of all boundary points is called the \textbf{boundary}\index{set!boundary} of $S$. \textbf{Notation}\vspace{-0.7cm} \begin{enumerate} \item Interior of $S$ is denoted by $\text{Int } S=\{z: z \text{ is an interior point of } S \}$ \item Exterior of $S$ is denoted by $\text{Ext } S=\{z: z \text{ is an exterior point of } S \}$ \item Boundary of $S$ is denoted by $\partial S=\{z: z \text{ is a boundary point of } S \}$ \end{enumerate} Considering the previous examples of neighbourhoods, let $S_1=B_1(0)$, $S_2=\overline{B}_{\frac{7}{8}}\left(-1-\sqrt{2}i\right)$ and $S_3=\overline{B}_{\frac{1}{2}}\left(2+\sqrt{3}i\right)\setminus \left\{2+\sqrt{3}i\right\}$. Thus, for $S_1$ we have: \begin{itemize} \item $\text{Int } S_1=B_1(0)$ \item $\text{Ext } S_1=\{z: |z|>1 \}$ \item $\partial S_1=\{z: |z|=1 \}$ \end{itemize} For $S_2$ we have: \begin{itemize} \item $\text{Int } S_2=B_{\frac{7}{8}}\left(-1-\sqrt{2}\right)$ \item $\text{Ext } S_2=\left\{z:|z-\left(-1-\sqrt{2}i\right)|>\frac{7}{8}\right\}$ \item $\partial S_2=\left\{z:| z-\left(-1-\sqrt{2}i\right)| =\frac{7}{8}\right\}$ \end{itemize} And finally, for $S_3$ we have: \begin{itemize} \item $\text{Int } S_3=B_{\frac{1}{2}}\left(2+\sqrt{3}\right)$ \item $\text{Ext } S_3=\left\{z:|z-\left(2+\sqrt{3}i\right)|>\frac{1}{2}\right\}$ \item $\partial S_3=\left\{z:|z-\left(2+\sqrt{3}i\right)|=\frac{1}{2}\right\}\cup\{0\}$ \end{itemize} %\vspace{-0.7cm} %\begin{center} % \begin{tabular}{ |c| c | c |c |} % \hline % & $S_1$ & $S_2$ & $S_3$ \\ \hline\hline % \textbf{Int} & $B_1(0)$ & $B_{\frac{7}{8}}\left(-1-\sqrt{2}\right)$& $B_{\frac{1}{2}}\left(2+\sqrt{3}\right)\setminus \{2+\sqrt{3}i\}$ \\ %\textbf{Ext} & $\{z: |z|>1 \}$ & $\left\{z:|z-\left(-1-\sqrt{2}i\right)|>\frac{7}{8}\right\}$ & $\left\{z:|z-\left(2+\sqrt{3}i\right)|>\frac{1}{2}\right\}$ \\ % \textbf{Bdy} & $\{z: |z|=1 \}$ & $\left\{z:|z-\left(-1-\sqrt{2}i\right)|=\frac{7}{8}\right\}$ & $\left\{z:|z-\left(2+\sqrt{3}i\right)|=\frac{1}{2}\right\}\cup\{0\}$ \\ % \hline % \end{tabular} %\end{center} A set $S$ is \textbf{open}\index{set!open} if for every $z\in S$, exits $\epsilon>0$ such that $$B_{\epsilon}(z)\subset S.$$ That is, $\text{Int } S= S$. A set $S$ is \textbf{closed}\index{set!closed} if it contains all of its boundary points, that is $$\partial S\subseteq S.$$ The set $\mathbb C$, together with the collection $\tau =\{S\subseteq \mathbb C: S \text{ is open} \}$ is a topological space, and this is expressed by the pair $\left(\mathbb C, \tau\right)$. The topological space $\left(\mathbb C, \tau\right)$ satisfies the following: \begin{enumerate} \item $\emptyset$ and $\mathbb C$ are open. \item Whenever two or more sets are open, then so is their union. \item Whenever sets $S_1$ and $S_2$ are open, then so is $S_1\bigcap S_2$. \end{enumerate} On the other hand, considering closed sets, we have the following properties: \begin{enumerate} \item $\emptyset$ and $\mathbb C$ are closed. \item Whenever two or more sets are closed, then so is their union. \item Whenever sets $S_1$ and $S_2$ are closed, then so is $S_1\bigcap S_2$. \end{enumerate} The set $\mathbb C$ is both open and closed since it has no boundary points. The \textbf{closure}\index{set!closure} of a set $S$ is the closed set consisting of all points in $S$ together with the boundary of $S$. In other words $$\overline{S}=S\cup \partial S.$$ An open set $S$ is \textbf{connected}\index{set!connected} if each pair of points $z_1$ and $z_2$ in it can be joined by a polygonal line, consisting of a finite number of line segments joined end to end, that lies entirely in $S$. \begin{figure}[h] \centering \begin{minipage}{.5\textwidth} \centering \includegraphics[width=0.7\linewidth]{circleone} \caption{The open set $|z| < 1$} \label{set001} \end{minipage}% \begin{minipage}{.5\textwidth} \centering \includegraphics[width=0.7\linewidth]{anulus} \caption{The open set $1 < |z| < 2$} \label{set002} \end{minipage} \end{figure} The open set $|z| < 1$ is connected. The annulus $1 < |z| < 2$ is open and it is also connected (see Figures \ref{set001} and \ref{set002}). A nonempty open set that is connected is called a \textbf{domain}\index{domain}. Note that any neighbourhood is a domain. A domain together with some, none, or all of its boundary points is referred to as a \textbf{region}\index{domain!region}.
