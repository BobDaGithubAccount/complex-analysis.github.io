Airy functions

Introduction

In the physical sciences, the Airy function (or Airy function of the first kind) $\text{Ai}(z)$ is a special function named after the British astronomer George Biddell Airy (1801â€“1892). The function $\text{Ai}(z)$ and the related function $\text{Bi}(z)$, are linearly independent solutions to the differential equation \begin{eqnarray}\label{airy} \frac{d^2y}{dz^2} - yz = 0 \end{eqnarray} written in the form $$y(z) = A \, \text{Ai}(z) + B\, \text{Bi}(z),$$ with $$\text{Ai}(z) = \frac{1}{3^{2/3}\Gamma\left(\frac{2}{3}\right)} {}_{0}F_{1} \left(;\frac{2}{3}, \frac{1}{9}z^3\right) - \frac{z}{3^{1/3} \Gamma\left(\frac{1}{3}\right)} {}_{0}F_{1} \left(;\frac{4}{3}, \frac{1}{9}z^3\right)$$ and $$\text{Bi}(z) = \frac{1}{3^{1/6}\Gamma\left(\frac{2}{3}\right)} {}_{0}F_{1}\left(;\frac{2}{3}, \frac{1}{9}z^3\right) + \frac{3^{1/6}z}{\Gamma\left(\frac{1}{3}\right)} {}_{0}F_{1} \left(;\frac{4}{3}, \frac{1}{9}z^3\right)$$ where $ _0F_1(;a;z )$ is a confluent hypergeometric limit function.

The differential equation defined in (\ref{airy}) is known as the Airy equation or the Stokes equation. This is the simplest second-order linear differential equation with a turning point (a point where the character of the solutions changes from oscillatory to exponential).
