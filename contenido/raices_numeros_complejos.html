Raíces de Números Complejos

Recordemos que si $z=x+iy$ es un número complejo diferente de zero, entonces se puede escribir en su forma polar como \[z=r(\cos \theta +i \,\text{sen } \theta)\] donde $r=\sqrt{x^2+y^2}$ y $\theta$ es el ángulo, en radianes, medido desde el eje positivo $x$ al rayo que conecta el origen con el punto $z$.

Ahora, la fórmula de De Moivre's establece que si $z=r(\cos \theta +i\,\text{sen } \theta)$ y $n$ es un entereo positivo, entonces \begin{eqnarray*} z^n=r^n(\cos n\theta+i\,\text{sen } n\theta). \end{eqnarray*}

Sea $w$ un número complejo. Usando la fórmula de De Moivre nos ayudará a resolver la ecuación $$z^n=w$$ para $z$ cuando $w$ es dado.

Supongamos que $w=r(\cos \theta +i\,\text{sen } \theta)$ y $z=\rho (\cos \psi +i\,\text{sen } \psi)$. Entonces De Moivre nos dice que $$z^n=\rho^n(\cos n\psi+i\,\text{sen } n\psi).$$ De aquí sigue que $$\rho^n=r=|w|$$ por la unicidad de la representación polar y además $$n\psi = \theta +k(2\pi),$$ donde $k$ es un entero. De esta forma

\[z=\sqrt[n]{r}\left[\cos\left(\frac{\theta}{n}+\frac{2k\pi}{n}\right)+i\,\text{sen }\left(\frac{\theta}{n}+\frac{2k\pi}{n}\right) \right].\]
Cada valor de $k=0,1,2,\ldots ,n-1$ nos da un valor diferente de $z$. Cualquier otro valor de $k$ simplemente repite uno de los valores de $z$ conrrespondiente a $k=0,1,2,\ldots ,n-1$. Por lo que hay exactamente $n$ raíces de un número complejo $z\neq 0$.

Usando la fórmula de Euler: $$e^{i\theta}=\cos \theta +i \,\text{sen } \theta,$$ el número complejo $z=r(\cos \theta +i\,\text{sen } \theta)$ se puede escribir también es su forma eponencial como $$z=re^{i\theta}$$

De esta manera, las $n$ raíces del número complejo $z\neq 0$ se pueden expresar como \begin{eqnarray}\label{expform} z=\sqrt[n]{r}\;\mbox{exp}\left[i\left(\frac{\theta}{n}+\frac{2k\pi}{n}\right)\right] \end{eqnarray} donde $k=0, 1, 2, \ldots , n-1$.

El applet siguiente muestra una representación geométrica de las $n$ ráices de un número complejo, hasta $n=10$. Mueve el punto rojo alrededor para cambiar el valor de $z$ o mueve los deslizadores.

Código

Usa el siguiente script en GeoGebra para que explorar o hacer tu propia versión. El símbolo # indica comentarios.

Z = 1 + ί

#Módulo de Z

r = abs(Z)

#Ángulo de Z

theta = atan2(y(Z), x(Z))

#Número de raíces

n = Deslizador(2, 10, 1, 1, 150, false, true, false, false)

#Grafica n-raíces
